# AgViz roadmap

## Milestone 0: repository setup

- Add `.github/copilot-instructions.md`
- Add `AGENTS.md`
- Add CI
- Add issue templates
- Add bootstrap prompt

## Milestone 1: browser-native MVP

- Vite React TypeScript scaffold
- GFA parser for `H`, `S`, `L`, `P`
- Internal assembly graph model

- File upload
- Node and edge inspector
- Unit tests

- Search by segment ID
- Filter by length
- Filter by coverage
- Layout selector
- Better graph statistics
- Export PNG/SVG
- `.gfa.gz` support

## Milestone 3: scale

- Web Worker parser
- Progressive loading
- Large-graph warnings
- Neighbourhood extraction
- Component filtering
- Subgraph export

## Milestone 4: experiments

- Graphviz/Wasm layout branch
- Pyodide branch
- agtools compatibility experiments
